Q 1 - The unit digit in 7153 is?

A - 6

B - 0

C - 7

D - 1

Answer : C

Explanation

7153 = (74)38 x 7
Now, unit digit of (74)38 = 1
Therefore, unit digit of 7153 = 1 x 7
= 7

Q 2 - If the number 3z4 is divisible by 6, then value of z =?

A - 0

B - 1

C - 6

D - 2

Answer : D

Explanation

6 = 3 x 2 i.e. divisible by 2
For divisibility: (3 + z + 4) must be divisible by 3
Therefore, z = 2

Q 3 - On dividing a number by 5, we get remainder as 3. What will be the remainder when the square of this number is divided by 5?

A - 3

B - 0

C - 1

D - 4

Answer : D

Explanation

Let number be p
Dividing p by 5, we get k as quotient and remainder as 3
p = 5k + 3 ->
p2 = (5k + 3)2 p = (25k2 + 30k + 9)
p = 5 (5k2 + 6k + 1) + 4
Therefore, we get 4 as remainder on dividing p2 by 5.

Q 4 - What will be the remainder when 17200is divided by 18?

A - 1

B - 3

C - 19

D - 8

Answer : A

Explanation

(17200 - 1200) is completely divisible by (x + a) when n is even.
= (17200 - 1200) is completely divisible by 18 i.e.(17 + 1)
= (17200 - 1) is completely divisible by 18.
Therefore, the remainder is 1.

Q 5 - 6 + 15 + 24 + 33 + ... + 123 = ?

A - 903

B - 603

C - 752

D - 690

Answer : A

Explanation

a = 6, d = 9, l = 123
Let number of terms be n
123 = a + (n - 1)d
123 = 6 + (n - 1)9
123 = 9n - 9
n = 14
Sn = n⁄2 (a + l)
= 14⁄2 (6 + 123)
= 7 x 129
= 903

Q 6 - What is the unit digit in in 365 x 659 x 771 ?

A - 1

B - 2

C - 4

D - 6

Answer : C

Explanation

  365 = (34)6 x 3 So Unit digit in 365 = Unit digit in 1 x 3 = 3  659 = (64)14 x 63 So Unit digit in 659 = Unit digit in 6 x 216 = 6  771 = (74)17 x 73 So Unit digit in 771 = Unit digit in 1 x 343 = 3  So Unit digit in 365 x 659 x 771 = Unit digit in 3 x 6 x 3 = 4.

A - 78

B - 68

C - 84

D - 66

Answer : A

Explanation
