# Chhattisgarh PSC Syllabus
This section has incorporated a comprehensive repository of Chhattisgarh Public Service Commission Syllabus that gives the details of Preliminary Test (Chhattisgarh PT) syllabus; Chhattisgarh Main Examination Syllabus; &amp; Personality Test details.

Based on the exam pattern, the presentation is well arranged. For the candidates’ convenience, the syllabus is summarized in a table. Lastly, based on the exams’ marks distribution, each exam (i.e. PT, Mains, &amp; Interview) has been also presented with respective marks (that it carries in the exam).

# Audience
The Chhattisgarh Public Service Commission Exam Syllabus is designed exclusively for the candidates preparing for the CHPSC exam.

# Prerequisites
There is as such no prerequisite; any candidate preparing for the Chhattisgarh Public Service Commission exam can take advantage of this.
