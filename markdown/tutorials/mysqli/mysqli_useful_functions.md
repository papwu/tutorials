# MySQLi - Useful Functions
Here is the list of all important MySQLi functions. Each function has been explained along with suitable example.

   * [mysqli::$affected_rows](/mysqli/mysqli_affected_rows.htm)  − It used to get the information about number of affected rows in a previous MySQL operation
   * [mysqli::autocommit](/mysqli/mysqli_autocommit.htm)  − It used to turn on or off auto-committing database modifications operation
   * [mysqli::begin_transaction](/mysqli/mysqli_begin_transaction.htm)  − It used to start a transaction
   * [mysqli::change_user](/mysqli/mysqli_change_user.htm)  − It used to change the user of the specified database connection
   * [mysqli::character_set_name](/mysqli/mysqli_character_set_name.htm)  - It returns the default character set for the database connection.
   * [mysqli::$client_info](/mysqli/mysqli_client_info.htm)  − It is used to get MySQL client info.
   * [mysqli::$client_version](/mysqli/mysqli_client_version.htm)  − It returns the MySQL client version as a string.
   * [mysqli::close](/mysqli/mysqli_close.htm)  − It closes a previously opened database connection.
   * [mysqli::commit](/mysqli/mysqli_commit.htm)  − It commits the current transaction.
   * [mysqli::$connect_errno](/mysqli/mysqli_connect_errno.htm)  − It returns the error code from last connect call.
   * [mysqli::connect_error](/mysqli/mysqli_connect_error.htm)  − It returns a string description of the last connect error.
   * [mysqli::__construct](/mysqli/mysqli_construct.htm)  − It used to open a new connection to the MySQL server
   * [mysqli::debug](/mysqli/mysqli_debug.htm)  − It used to performs debugging operations
   * [mysqli::dump_debug_info](/mysqli/mysqli_dump_debug_info.htm)  − It is used to dump debugging information into the log
   * [mysqli::$errno](/mysqli/mysqli_errno.htm)  − It returns the error code for the most recent function call
   * [mysqli::$error_list](/mysqli/mysqli_error_list.htm)  − It returns a list of errors from the last command executed
   * [mysqli::$error](/mysqli/mysqli_error.htm)  − It returns a string description of the last error
   * [mysqli::$field_count](/mysqli/mysqli_field_count.htm)  − It returns the number of columns for the most recent query
   * [mysqli::get_charset](/mysqli/mysqli_error_get_charset.htm)  − It returns a character set object
   * [mysqli::get_client_info](/mysqli/mysqli_get_client_info.htm)  − It is used to get MySQL client info
   * [mysqli::mysqli_get_client_stats](/mysqli/mysqli_get_client_stats.htm)  − It returns client per-process statistics
   * [mysqli::mysqli_get_client_version](/mysqli/mysqli_get_client_version.htm)  − It returns the MySQL client version as an integer
   * [mysqli::get_connection_stats](/mysqli/mysqli_get_connection_stats.htm)  − It returns statistics about the client connection
   * [mysqli::$host_info](/mysqli/mysqli_host_info.htm)  − It returns a string representing the type of connection used
   * [mysqli::$protocol_version](/mysqli/mysqli_protocol_version.htm)  − It returns the version of the MySQL protocol used
   * [mysqli::$server_info](/mysqli/mysqli_server_info.htm)  − It returns the version of the MySQL server
   * [mysqli::$server_version](/mysqli/mysqli_server_version.htm)  − It returns the version of the MySQL server as an integer
   * [mysqli::get_warnings](/mysqli/mysqli_get_warnings.htm)  − It is used to get result of SHOW WARNINGS
   * [mysqli::$info](/mysqli/mysqli_info.htm)  − It retrieves information about the most recently executed query
   * [mysqli::$insert_id](/mysqli/mysqli_insert_id.htm)  − It returns the auto generated id used in the latest query
   * [mysqli::kill](/mysqli/mysqli_kill.htm)  − It asks the server to kill a MySQL thread
   * [mysqli::more_results](/mysqli/mysqli_more_results.htm)  − It used to check if there are any more query results from a multi query
   * [mysqli::multi_query](/mysqli/mysqli_multi_query.htm)  − It performs a query on the database
   * [mysqli::next_result](/mysqli/mysqli_next_result.htm)  − It prepares next result from multi_query
   * [mysqli::options](/mysqli/mysqli_options.htm)  − It used to set options
   * [mysqli::ping](/mysqli/mysqli_ping.htm)  − It is used to ping a server
   * [mysqli::poll](/mysqli/mysqli_poll.htm)  − It used to poll connections
   * [mysqli::prepare](/mysqli/mysqli_prepare.htm)  − It used to prepare an SQL statement for execution.
   * [mysqli::query](/mysqli/mysqli_query.htm)  − It is used to perform a query on the database.
   * [mysqli::real_connect](/mysqli/mysqli_real_connect.htm)  − It opens a connection to a mysql server.
   * [mysqli::real_escape_string](/mysqli/mysqli_real_escape_string.htm)  − It escapes special characters in a string for use in an SQL statement
   * [mysqli::real_query](/mysqli/mysqli_real_query.htm)  − It is used to execute an SQL query.
   * [mysqli::real_async_query](/mysqli/mysqli_real_async_query.htm)  − It is used to get result from async query
   * [mysqli::refresh](/mysqli/mysqli_refresh.htm)  − It used to refresh an SQL statement for execution.
   * [mysqli::release_savepoint](/mysqli/mysqli_release_savepoint.htm)  − It removes the named savepoint from the set of savepoints of the current transaction
   * [mysqli::rollback](/mysqli/mysqli_rollback.htm)  − It rolls back current transaction
   * [mysqli::rpl_query_type](/mysqli/mysqli_rpl_query_type.htm)  − It returns RPL query type
   * [mysqli::select_db](/mysqli/mysqli_select_db.htm)  − It is used to selects the default database for database queries
   * [mysqli::send_query](/mysqli/mysqli_send_query.htm)  − It is used to send the query and return
   * [mysqli::set_charset](/mysqli/mysqli_set_charset.htm)  − It sets the default client character set
   * [mysqli::set_local_infile_default](/mysqli/mysqli_set_local_infile_default.htm)  − It is used to unsets user defined handler for load local infile command
   * [mysqli::set_local_infile_handler](/mysqli/mysqli_set_local_infile_handler.htm)  − It is used to set callback function for LOAD DATA LOCAL INFILE command
   * [mysqli::$sqlstate](/mysqli/mysqli_sqlstate.htm)  − It returns the SQLSTATE error from previous MySQL operation.
   * [mysqli::ssl_set](/mysqli/mysqli_ssl_set.htm)  − It is used for establishing secure connections using SSL
   * [mysqli::stat](/mysqli/mysqli_stat.htm)  − It is used to set callback function for LOAD DATA LOCAL INFILE command
   * [mysqli::stmt_init](/mysqli/mysqli_stmt_init.htm)  − It is used to initialize a statement and returns an object for use with mysqli_stmt_prepare.
   * [mysqli::mysqli::$thread_id](/mysqli/mysqli_thread_id.htm)  − It returns the thread ID for the current connection
   * [mysqli::thread_safe](/mysqli/mysqli_thread_safe.htm)  − It returns whether thread safety is given or not.
   * [mysqli::use_result](/mysqli/mysqli_use_result.htm)  − It is used to initiate a result set retrieval
   * [mysqli::$warning_count](/mysqli/mysqli_warning_count.htm)  − It returns the number of warnings from the last query for the given link
